Unit 2 · The Fretboard

Intervals as shapes

Here's the fretboard's best secret: a distance between two notes is a picture your hand can hold — and the picture works from any starting note, anywhere on the neck.

Before you start Note names on the neck (previous page) and the word interval = distance between two notes.

The idea

Because the strings are tuned evenly (five frets apart, every pair), a given musical distance always makes the same visual shape. A fifth is always "one string up, two frets over" — whether you start on E, on B♭, at the 3rd fret or the 10th. Learn five shapes and you can build most of the bass lines in this course without hunting for names first.

See them, hear them

Pick a shape — then tap any fret on the E or A string to move the root there. The shape follows.

The five (plus one) shapes that matter first

IntervalShape from the rootSounds like / used for
Fifth (7 semitones)up 1 string, up 2 fretspower, stability — the root's best friend in every style
Fourth (5 semitones)up 1 string, same fretthe next-string note; smooth motion between roots
Octave (12)up 2 strings, up 2 fretsthe same note, higher — disco/funk jumps
Major third (4)up 1 string, back 1 fretthe "happy" color note of major chords
Minor third (3)up 1 string, back 2 frets (or same string +3)the "sad" color note of minor chords
Minor seventh (10)up 2 strings, same fretthe bluesy tension note of 7th chords

Down-a-fifth deserves a mention too: same fret, one string down lands a fifth below the root — the "low fifth" country and rock lines bounce off constantly.

The details (open as much as you like)

Why shapes beat letter names (at first)

Names tell you what a note is; shapes tell your hand where relationships live. Real bass lines are built from relationships — "root, fifth, octave, flat-seven" — so a movable shape vocabulary means one lick learned = twelve keys owned. The names still matter (you'll call shapes BY their interval names forever), which is why the last page taught them first.

The one caveat: crossing to the G string

All shapes here are true everywhere on a 4-string bass in standard tuning because every string pair is a fourth apart. Guitarists' shapes warp when they cross their B string — if you ever move patterns between the instruments, that's the difference you're feeling.

Practice

Shapes go into your hands by moving them around, not by staring. Say the interval name out loud on every pair.
